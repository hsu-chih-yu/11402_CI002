# LeetCode 724 - Find Pivot Index

## 題目描述
找出陣列中的 Pivot Index。中心索引的定義是：該索引 **左側** 所有數字之和等於 **右側** 所有數字之和。
- 如果中心索引位於陣列最左端，則左側和為 0。
- 如果中心索引位於陣列最右端，則右側和為 0。
- 若有多個中心索引，回傳 **最左邊** 的那一個；若不存在，回傳 -1。

[題目連結](https://leetcode.com/problems/find-pivot-index/)

## 學習重點
- [x] 理解 **前綴和** 與 **後綴和** 的動態轉移。
- [x] 掌握 $O(N)$ 時間複雜度的演算法設計，避免 $O(N^2)$ 的重複計算。
- [x] 處理邊界條件：當索引在 0 或 $n-1$ 時的和。



## 解法思路：動態總和平衡法
這份實作採用了「先求總和，再動態加減」的策略：

1. **初始化右側總和 (`right`)**：
   - 先用一個 `for` 迴圈計算從 `nums[1]` 到 `nums[len-1]` 的所有數字總和。
   - 此時 `left` 初始化為 0。
2. **線性掃描與比較**：
   - 遍歷每一個索引 `i`。
   - **檢查**：如果目前的 `left == right`，代表 `i` 就是中心索引，立即回傳。
3. **動態更新**：
   - 如果不相等，準備移向下一位：
   - 將目前的 `nums[i]` 加入 `left`。
   - 將下一個數字 `nums[i+1]` 從 `right` 中扣除（需判斷 `i < len - 1` 防止溢位）。

## 程式碼實作 (Code Structure)
```cpp
#include <vector>

using namespace std;

class Solution {
public:
    int pivotIndex(vector<int>& nums) {
        int len = nums.size();

        int right = 0, left = 0;

        // 初始化：計算除了第一個元素以外的所有右側總和
        for(int i = 1; i < len; i++){
            right += nums[i];
        }

        // 遍歷每一個可能的中心索引
        for(int i = 0; i < len; i++){
            // 判斷左側是否等於右側
            if(left == right){
                return i;
            }

            // 動態更新：準備將中心點往右移一位
            left += nums[i]; // 當前點變為左側的一部分
            
            if(i < len - 1) {
                right -= nums[i + 1]; // 下一個點從右側總和中移出
            }
        }

        return -1;
    }
};
```

## 複雜度分析
- **時間複雜度**: $O(N)$
  雖然有兩個 `for` 迴圈，但它們是先後執行的，總共只遍歷了兩次陣列。
- **空間複雜度**: $O(1)$
  僅使用了幾個整數變數，與陣列大小無關。



## 筆記與心得
- **更簡潔的思維**：另一種常見寫法是先求出「全體總和 `total`」，則在任何索引 `i`，其右側和必為 `total - left - nums[i]`。這樣可以少寫一個 `if(i < len - 1)` 的判斷。
- **穩定性**：目前的寫法由左往右搜尋並在第一時間回傳，完美符合題目要求「回傳最左邊索引」的規則。
